# AI Fundamentals for BlackRoad OS Developers

**Duration:** 4 weeks | **Level:** Beginner-Intermediate | **Prerequisites:** Basic Python

## Modules

- Week 1: Local LLM Foundations — Ollama, prompt engineering, model selection
- Week 2: Agent Architecture — BlackRoad agents, memory systems, multi-agent coordination
- Week 3: Edge AI on Raspberry Pi — Hardware setup, model quantization, world-engine
- Week 4: Building Your Own Agent — Design patterns, br CLI integration, fleet deployment
