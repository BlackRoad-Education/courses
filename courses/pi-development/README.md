# Raspberry Pi Development with BlackRoad OS

**Duration:** 3 weeks | **Level:** Intermediate | **Prerequisites:** Linux basics, Python

## Modules

- Week 1: Pi Setup + BlackRoad Fleet — Raspberry Pi OS, Pi agent install, br pi commands
- Week 2: Autonomous World Generation — world-engine.py, custom themes, git push automation
- Week 3: Advanced IoT — GPIO sensors, OTA updates, multi-Pi coordination

## Hardware
- Raspberry Pi 4B (4GB+ recommended)
- 32GB+ microSD card
- Official 5V 3A power supply
